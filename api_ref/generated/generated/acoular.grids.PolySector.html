PolySector¶

class acoular.grids.PolySector¶

Class for defining a polygon sector.

Can be used for 2D Grids for definining a polygon sector.

contains(pos)¶

Queries whether the coordinates in a given array lie within the ploygon sector. If no coordinate is inside, the nearest one to the rectangle center is returned if default_nearest is True.

Parameters:
posarray of floats

Array with the shape 3x[number of gridpoints] containing the grid positions
