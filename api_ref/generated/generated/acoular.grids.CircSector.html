CircSector¶

class acoular.grids.CircSector¶

Bases: SingleSector

Class for defining a circular sector.

Can be used for 2D Grids for definining a circular sector or for 3D grids for a cylindrical sector parallel to the z-axis.

contains(pos)¶

Queries whether the coordinates in a given array lie within the circular sector. If no coordinate is inside, the nearest one outside is returned if default_nearest is True.
