PNoiseGenerator¶

class acoular.signals.PNoiseGenerator¶

Bases: SignalGenerator

Pink noise signal generator.

Simulation of pink noise is based on the Voss-McCartney algorithm. Ref.:

  • S.J. Orfanidis: Signal Processing (2010), pp. 729-733

  • online discussion: http://www.firstpr.com.au/dsp/pink-noise/

The idea is to iteratively add larger-wavelength noise to get 1/f characteristic.

seed = Int(0, desc='random seed value')¶

Seed for random number generator, defaults to 0. This parameter should be set differently for different instances to guarantee statistically independent (non-correlated) outputs.

depth = Int(16, desc='octave depth')¶

“Octave depth” – higher values for 1/f spectrum at low frequencies, but longer calculation, defaults to 16.

signal()¶

Deliver the signal.
