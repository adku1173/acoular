PointSourceDipole¶

class acoular.sources.PointSourceDipole¶

Bases: PointSource

Class to define a fixed point source with an arbitrary signal and dipole characteristics via superposition of two nearby inversely phased monopoles. This can be used in simulations.

The output is being generated via the result() generator.

direction = Tuple((0.0, 0.0, 1.0), desc='dipole orientation and distance of the inversely phased monopoles')¶

Vector to define the orientation of the dipole lobes. Its magnitude governs the distance between the monopoles (dist = [lowest wavelength in spectrum] x [magnitude] x 1e-5). Note: Use vectors with order of magnitude around 1.0 or less for good results.

result(num=128)¶

Python generator that yields the output at microphones block-wise.

Parameters:
numinteger, defaults to 128

This parameter defines the size of the blocks to be yielded (i.e. the number of samples per block) .

Returns:
